# Cisco Packet Tracer Labs

This repository contains hands-on networking labs built using Cisco Packet Tracer as part of my journey toward becoming a Network / Data Center Engineer.

---

## Labs Included

- SOHO Network (Wired and Wireless)
- DHCP Configuration
- NAT (Network Address Translation)
- Network Testing (Ping and IPConfig)
- FTP Services
- Telnet and SSH
- Routing between LANs

---

## Skills Demonstrated

- Network topology design  
- IP addressing and subnetting  
- DHCP configuration  
- NAT implementation  
- Basic routing and switching  
- Wireless network setup  
- Network troubleshooting using ping and ipconfig  

---

## Tools Used

- Cisco Packet Tracer  

---

## Lab Structure

Each lab is organized in separate folders:

01-soho-network/  
02-dhcp-configuration/  
03-nat-configuration/  

Each folder contains:
- Packet Tracer (.pkt) file  
- README.md (Lab explanation)  
- Screenshots (where applicable)
